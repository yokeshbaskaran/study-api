# API

What is API?

- API = Application Programming Interface.
- API is a way for two applications to talk to each other.

## How API works in real time?

1. Used Google Maps inside a food delivery app? --> That's an API showing the map.
2. Checked the weather on your phone? -->The app used a Weather API to get the info.
3. Logged into a website using your Google or Facebook account? --> That's an API handling the login.

## Some Network Terms:

-> Understanding API with a Restaurant Analogy

1. Client = The one who makes the request (like a browser or app)
   - Just like a customer placing an order in a restaurant.

2. API = The middleman that passes the request to the server and brings back the response.
   - Like a waiter who takes your order to the kitchen and brings your food.

3. Server = Where all the processing happens (the backend)
   - Like the kitchen, where your order gets prepared.

## Types of APIs:

=> Real-World Example: Uploading a Photo on Instagram

1. Uses Hardware API to access your camera
2. Uses Software API to apply filters, make your image appear better
3. Uses Web API to upload the photo from your Phone to Instagram's server
